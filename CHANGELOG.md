- Fish shell config with Dracula theme and modern tool aliases
- PowerShell 7 profile (Windows mirror of fish config)

- Clipboard integration (clip/clipfile/clipwd)
- Directory bookmarks with fzf picker
- Three-layer help system: shelp, tools, navi
- Conventional commits + auto-versioning GitHub Action
